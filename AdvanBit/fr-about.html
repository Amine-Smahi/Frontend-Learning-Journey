Qu'est-ce que AdvanBit?


AdvanBit est un Bitcoin PTC (Paid-to-Click) site publicitaire qui vous permet de gagner bitcoin librement, tout simplement pour visiter des sites Web.

Comment puis-je payé?


Vous pouvez être payé pour l'affichage des sites Web sur AdvanBit. Vous devez voir le site jusqu'à ce que la minuterie dans le coin supérieur gauche compte vers le bas à 0. Une fois que cela se produit, un bouton apparaîtra marqué "Pay Me". Une fois que vous cliquez sur le bouton, vous serez présenté avec un captcha pour répondre. Ceci afin d'éviter les robots qui abusent du service. Si vous entrez le code captcha correctement, vous serez payé instantanément sur votre compte MicroWallet, et la prochaine annonce disponible (le cas échéant) chargerez

Combien dois-je payé pour l'affichage d'une annonce?


les paiements des utilisateurs varient en fonction du budget de la campagne publicitaire lui-même. Les campagnes qui vous obligent à voir pendant 60 secondes vont payer vous plus d'une annonce demandant une seconde vue 10 par exemple.
